# Freddy's Faz-Pizza Builder – Engineer Specials 🍕🐻

A completely useless but highly entertaining **FNAF-themed pizza maker** built for the coding club creative challenge.

Make cursed pizzas by stacking toppings, hit "Bake!", and get rewarded with terrible engineering/programmer jokes twisted with Five Nights at Freddy's lore.  
Zero utility. Maximum Fazbear vibes. Pure chaos.

## Features
- Build your pizza with 5 cursed toppings:
  - Pepperoni = Freddy's favorite
  - Cheese overload = Chica's dream
  - Mushrooms = Bonnie's secret
  - Olives = Foxy's "hook"
  - Sausage = Springtrap's twisted special
- Stack toppings as much as you want (Chica would approve)
- "Bake!" button unlocks a random engineering/FNAF pun joke card
- Retro Freddy Fazbear's Pizza aesthetic + CRT/scanline filter
- Silly extras: multiple toppings allowed, reset button, localStorage "most cursed pizza" memory
- Dark horror-comedy theme with neon accents

## Tech Stack
- **React** + **Vite** (fast & modern)
- **Tailwind CSS** (for that quick retro styling)
- **Framer Motion** (smooth topping animations & card reveals)
- Pure frontend — no backend, no login, just vibes
